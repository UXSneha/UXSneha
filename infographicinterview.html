Infographic Creation Challenge: Improving the Venngage User Experience

Introduction

This case study explores the process of creating an infographic for a UX design job application at Venngage. The goal was to identify areas for improvement in the Venngage website's user experience (UX) and showcase them in a visually appealing infographic using Venngage itself.

The Challenge was twofold:

  • Limited Time: As a first-time UX designer applying for a job, a tight deadline of 2-4 hours was set for completing the infographic.
  • Dual Role: Having no prior experience with Venngage, acting as both a new user and a UX designer to evaluate the platform's UX.

HERE'S MY DESIGN PROCESS:

  • TIME MANAGEMENT: Sunday evening marked the start of the project, with a target completion within 2-4 hours.

  • TASK BREAKDOWN:
    • Learn Venngage basics.
    • Analyze the Venngage website for UX improvement opportunities.
    • Design and create the infographic using Venngage.

User Research (Playing the User)

To understand the user experience, I adopted a new user persona and performed the following actions:

  • Browsed the Venngage website.
  • Took screenshots of areas needing improvement.
  • Identified pain points and potential solutions.

Wireframing (Using Figma)

The following steps were taken to create wireframes for the proposed UX improvements:

  • Used Figma as the wireframing tool.
  • Used the screenshots as a base for the wireframes.
  • Visually depicted the proposed UX improvements on top of the screenshots.

Design and Implementation (Using Venngage)

Here's how the infographic was designed and created using Venngage:

  • Downloaded and reviewed the modified screenshots.
  • Explored Venngage templates for a suitable fit.
  • Considered the target audience (users seeking quick results with clear understanding of Venngage).
  • Prioritized a colorful, informative design to convey the intended message.
  • Selected a template that effectively accommodated the information.

Tools Used

  • Figma (for wireframing UX improvements)
  • Venngage (for infographic creation)

Lessons Learned

This project provided valuable learning experiences:

  • Gained valuable experience exploring the Venngage platform.
  • Demonstrated quick learning and adaptation to Venngage's functionalities.
  • Highlighted the importance of considering user experience in every digital interaction.
